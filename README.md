# Python-Calculator-With-PyQt-Prompts
A simple calculator written in Python with prompts and messages displayed using PyQt6
instead of the normal input() and print() statements.

## Tools Used

| Tool    |  Version |
|:--------|---------:|
| Python  |   3.13.0 |
| PyQt6   |    6.7.1 |
| VSCode  |   1.94.2 |
| PyCharm | 2024.2.4 |
